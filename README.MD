# 🐾 Paw Patrol Tracker 🐶

Keep track of your favorite Paw Patrol pups, missions, and adventures—all in one fun app!

---

## ✨ Features

- 🐕 Track characters
- 📋 Add missions & activities
- 👀 View and update saved data
- 💖 Super simple and easy to use

---

## 🚀 Installation

1. Clone the repo:
git clone https://github.com/kskenderaj/Paw-Patrol-Tracker.git

2. Open the project in your favorite IDE

3. Build and run the app on your device or emulator

---

## 🎮 How to Use

1. Launch the app
2. Add your favorite Paw Patrol characters
3. Create missions or activities
4. Save and view your tracking anytime
